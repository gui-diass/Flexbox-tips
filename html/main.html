O modelo que vamos aprender é o CSS Flexible Box Layout, mais conhecido como FlexBox.

O que é flexbox?

FlexBox é um módulo de caixa flexível, cujo módulo tem como objetivo alinhar, distribuir o espaço entre os contêiners. Ou seja o layout flexível permite que os elementos responsivos dentro de um contêiner sejam organizados automaticamente, dependendo do tamanho da tela.
